The *Vinaya Piṭaka* is the first section of the *Pāli Canon*. It contains the code of conduct for {% include tag-link.html text="the monks and nuns" tag="monastic" %} of the {% include tag-link.html text="Theravada tradition" tag="theravada" %} along with many entertaining stories from the early sangha.
